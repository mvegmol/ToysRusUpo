## Ejecutar el proyecto

- cambiar el nombre del archivo .env.example a .env
- composer install
- php artisan key:generate
- php artisan migrate
- php artisan db:seed
- npm install bootstrap --save-dev
- npm install sass --save-dev
- npm install bootstrap-icons
